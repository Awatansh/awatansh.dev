# Social Media Logos

Place your logo images here:

- `github-logo.png` - GitHub logo
- `linkedin-logo.png` - LinkedIn logo
- `twitter-logo.png` - Twitter/X logo
- `instagram-logo.png` - Instagram logo
- `facebook-logo.png` - Facebook logo
- `default-logo.png` - Default fallback logo

## Image Requirements

- Format: PNG or SVG
- Size: Recommended 512x512px or higher
- Background: Transparent preferred
- Style: Will be converted to monochrome (green theme) via CSS filters

## Notes

The CSS applies the following filters to match the terminal theme:

- `filter: brightness(0) invert(1)` - Converts logos to white
- `opacity: 0.8` - Slightly transparent
- On hover: `opacity: 1` and `scale(1.1)` for interaction

The logos are displayed in 36x36px circular containers with 6px padding.
